⚡ QUICK ACTIVATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AUTOMATIC METHOD (EASIEST):
────────────────────────────────────────────────────────────────────
Run this script from your project folder:

  $ ~/Project_Zenitsu_Theme/activate_thunder_theme.sh

This will:
  1. Activate the Zenitsu-Peony layout
  2. Apply the Thunder Breathing style
  3. Reload Waybar automatically

MANUAL METHOD:
────────────────────────────────────────────────────────────────────
Step 1 - Activate Layout:
  • Press: SUPER + ALT + B
  • Select: "⚡ [BOT] Zenitsu-Peony"
  • Press: Enter

Step 2 - Apply Style:
  • Press: SUPER + CTRL + B
  • Select: "⚡ [Zenitsu] Thunder Breathing"
  • Press: Enter

Step 3 - Reload:
  • Press: SUPER + ALT + R

MODIFY COLORS:
────────────────────────────────────────────────────────────────────
Edit the CSS file:
  $ nano ~/.config/waybar/style/⚡\ [Zenitsu]\ Thunder\ Breathing.css

Look for the @define-color section at the top:
  @define-color haori_orange #FFA237;
  @define-color thunder_yellow #FFE66D;
  @define-color lightning_white #FFFFFF;
  @define-color electric_blue #3498DB;

CHANGE ICONS:
────────────────────────────────────────────────────────────────────
Edit the modules file:
  $ nano ~/.config/waybar/ZenitsuModules

Find the module you want to change and replace the icon.
Use Nerd Fonts: https://www.nerdfonts.com/cheat-sheet

ADJUST LAYOUT:
────────────────────────────────────────────────────────────────────
Edit the main config:
  $ nano ~/.config/waybar/configs/⚡\ [BOT]\ Zenitsu-Peony

Modify "modules-left", "modules-center", or "modules-right"
to rearrange the module positions.

WAYBAR NOT UPDATING:
────────────────────────────────────────────────────────────────────
Try a hard reload:
  $ pkill waybar && waybar &

ICONS NOT SHOWING:
────────────────────────────────────────────────────────────────────
Ensure Nerd Fonts are installed:
  $ pacman -Qs nerd-fonts

COLORS TOO BRIGHT/DIM:
────────────────────────────────────────────────────────────────────
Adjust the alpha values in the CSS:
  background: alpha(@dark_bg, 0.7);  ← Change 0.7 to 0.5-0.9

THEME NOT IN MENU:
────────────────────────────────────────────────────────────────────
Verify files exist:
  $ ls ~/.config/waybar/configs/ | grep Zenitsu
  $ ls ~/.config/waybar/style/ | grep Zenitsu

Use the automatic activator:
  $ ~/Project_Zenitsu_Theme/activate_thunder_theme.sh
