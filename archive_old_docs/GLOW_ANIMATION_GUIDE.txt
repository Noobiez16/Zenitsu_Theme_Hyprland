📋 OVERVIEW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

This enhancement adds a constant, electrifying pulsing glow effect to your
Waybar, perfectly capturing the essence of Zenitsu's Thunder Breathing 
technique. The animation creates a living, breathing panel that pulses with
electric energy while maintaining full visibility and functionality.

WHAT IT DOES:
  ⚡ Adds a vibrant yellow glow that pulses constantly (2-second cycle)
  ⚡ Animates border colors between Haori Orange and Thunder Yellow
  ⚡ Enhances active workspaces with thunder effects
  ⚡ Creates special animations for battery charging
  ⚡ Adds subtle electric pulse to the clock
  ⚡ Makes the power button pulse with warning energy
  ⚡ Maintains all existing Thunder Breathing icons and colors

WHAT IT DOESN'T CHANGE:
  ✓ Layout stays exactly the same (Peony layout preserved)
  ✓ All icons remain unchanged
  ✓ Text colors and readability maintained
  ✓ Module positions unchanged
  ✓ Original style files preserved

⚡ INSTALLATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Run the Installer Script
────────────────────────────────────────────────────────────────────
  $ cd ~/Project_Zenitsu_Theme
  $ ./apply_glow_animation.sh

  This will:
    • Create a new style: "⚡ [Zenitsu] Thunder Glow"
    • Add all glow animations to the CSS
    • Create a quick activation script
    • Preserve your original Thunder Breathing style

STEP 2: Activate the Glow Animation
────────────────────────────────────────────────────────────────────
  Quick Method:
    $ ~/Project_Zenitsu_Theme/activate_glow.sh

  Manual Method:
    1. Press: SUPER + CTRL + B
    2. Select: "⚡ [Zenitsu] Thunder Glow"
    3. Press: Enter

STEP 3: Enjoy!
────────────────────────────────────────────────────────────────────
  Your Waybar will now pulse with constant electric energy!

🎨 ANIMATION DETAILS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

MAIN PANEL GLOW (zenitsu-glow):
────────────────────────────────────────────────────────────────────
  Duration: 2 seconds per cycle
  Effect: Triple-layered glow effect
    • Outer glow: Vibrant yellow (#f9ff5e)
    • Middle glow: Haori Orange (#FFA237)
    • Inner glow: Subtle yellow highlight
  Timing: ease-in-out (smooth acceleration and deceleration)
  Loop: Infinite (constant pulsing)

BORDER COLOR ANIMATION (zenitsu-border-glow):
────────────────────────────────────────────────────────────────────
  Duration: 2 seconds (synchronized with main glow)
  Effect: Animated color transitions
    • Top border: Haori Orange ↔ Vibrant Yellow
    • Bottom border: Thunder Yellow ↔ Haori Orange
  Creates a living, breathing border effect

ACTIVE WORKSPACE THUNDER (workspace-thunder):
────────────────────────────────────────────────────────────────────
  Duration: 1.5 seconds per cycle
  Effect: Enhanced thunder glow on active workspace
  Colors: Thunder Yellow + Haori Orange
  Purpose: Makes active workspace stand out dramatically

BATTERY CHARGING (battery-thunder-charge):
────────────────────────────────────────────────────────────────────
  Duration: 1.8 seconds per cycle
  Effect: Intense pulsing during charging
  Colors: Vibrant yellow with orange accents
  Purpose: Visual feedback that battery is charging

CLOCK PULSE (clock-electric-pulse):
────────────────────────────────────────────────────────────────────
  Duration: 3 seconds per cycle (slower, more subtle)
  Effect: Gentle electric pulse
  Purpose: Subtle time-related energy visualization

POWER BUTTON (power-warning-pulse):
────────────────────────────────────────────────────────────────────
  Duration: 2.5 seconds per cycle
  Effect: Attention-grabbing pulse
  Colors: Thunder Yellow + Red (warning)
  Purpose: Makes power options clearly visible

🎛️  CUSTOMIZATION OPTIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ADJUST ANIMATION SPEED:
────────────────────────────────────────────────────────────────────
Edit: ~/.config/waybar/style/⚡ [Zenitsu] Thunder Glow.css

Find the animation line:
  animation: zenitsu-glow 2s ease-in-out infinite;

Change "2s" to adjust speed:
  • Faster pulse: 1s or 1.5s
  • Slower pulse: 3s or 4s

ADJUST GLOW INTENSITY:
────────────────────────────────────────────────────────────────────
Find the @keyframes zenitsu-glow section.

At 50% (peak intensity):
  box-shadow: 0 0 16px 6px rgba(249, 255, 94, 1.0);
                    ↑      ↑
              Increase for   Increase for
              more spread    more intensity

Adjust the values:
  • More intense: Increase blur (16px → 24px)
  • Larger glow: Increase spread (6px → 10px)
  • Brighter: Increase opacity (1.0 → 1.0 is max)

DISABLE SPECIFIC ANIMATIONS:
────────────────────────────────────────────────────────────────────
To disable just the main panel glow but keep others:

Find: window#waybar {
Remove or comment out the animation line:
  /* animation: zenitsu-glow 2s ease-in-out infinite; */

To disable workspace animation:
  /* #workspaces button.active { animation: ... } */

CHANGE GLOW COLORS:
────────────────────────────────────────────────────────────────────
Replace rgba(249, 255, 94, X) with your preferred color.

Current: #f9ff5e (Vibrant Thunder Yellow)
Alternatives:
  • Pure yellow: rgba(255, 255, 0, X)
  • Orange glow: rgba(255, 162, 55, X)
  • Blue electric: rgba(52, 152, 219, X)

YOU HAVE THREE WAYBAR STYLES AVAILABLE:

1. ⚡ [Zenitsu] Thunder Breathing (No animations)
   • Original Thunder themed style
   • Static colors and effects
   • Best for: Maximum performance

2. ⚡ [Zenitsu] Thunder Glow (With animations)
   • Constant pulsing glow effects
   • Animated borders and elements
   • Best for: Maximum visual impact

3. Original styles (Catppuccin, etc.)
   • System default styles
   • Available in the style menu

TO SWITCH STYLES:
────────────────────────────────────────────────────────────────────
  1. Press: SUPER + CTRL + B
  2. Navigate with arrow keys
  3. Press Enter on your choice
  4. Waybar reloads automatically

QUICK SHORTCUTS:
────────────────────────────────────────────────────────────────────
  Activate Thunder Glow:
    $ ~/Project_Zenitsu_Theme/activate_glow.sh

  Activate Thunder Breathing (no glow):
    $ ~/Project_Zenitsu_Theme/activate_thunder_theme.sh

ANIMATIONS NOT VISIBLE:
────────────────────────────────────────────────────────────────────
  1. Verify the glow style is active:
     $ readlink ~/.config/waybar/style.css
     Should show: ...Thunder Glow.css

  2. Reload Waybar:
     $ pkill waybar && waybar &

  3. Check for CSS errors:
     $ waybar 2>&1 | grep -i error

GLOW TOO SUBTLE:
────────────────────────────────────────────────────────────────────
  Edit the CSS and increase values in @keyframes zenitsu-glow:
    • Increase blur radius: 16px → 24px
    • Increase spread: 6px → 10px
    • Add more layers of box-shadow

GLOW TOO INTENSE:
────────────────────────────────────────────────────────────────────
  • Reduce opacity: 1.0 → 0.7 or 0.8
  • Decrease blur radius: 16px → 12px
  • Use Thunder Breathing style (no animation)

WAYBAR WON'T START:
────────────────────────────────────────────────────────────────────
  1. Switch to default style temporarily:
     $ ln -sf ~/.config/waybar/style/[Catppuccin]\ Latte.css \
              ~/.config/waybar/style.css
     $ waybar &

  2. Check for syntax errors in Thunder Glow.css

  3. Reinstall the glow animation:
     $ ~/Project_Zenitsu_Theme/apply_glow_animation.sh
