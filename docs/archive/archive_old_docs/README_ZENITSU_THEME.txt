WHAT IS THIS?
───────────────────────────────────────────────────────────────────
A comprehensive, high-fidelity theming system that transforms your
Hyprland environment to match Zenitsu Agatsuma's iconic aesthetic
from Demon Slayer. Features hard-coded color accuracy, custom
terminal logos, and automated installation.

QUICK START (3 STEPS)
───────────────────────────────────────────────────────────────────
1. Gather assets (wallpaper, logos) - See guides below
2. Run: ./apply_zenitsu_theme.sh
3. Reboot: sudo reboot

Done! Your desktop now embodies Thunder Breathing. ⚡

TROUBLESHOOTING QUICK REFERENCE
═══════════════════════════════════════════════════════════════════

"Missing dependencies"
  → yay -S swww wallust fastfetch

"No wallpaper found"
  → ls ~/Pictures/zenitsu_theme/wallpapers/
  → Download wallpaper using ASSET_REFERENCE_GUIDE.txt

"Custom logo not found"
  → Follow logo creation tutorial in ASSET_REFERENCE_GUIDE.txt
  → Verify: ls ~/Pictures/zenitsu_theme/logos/zenitsu_arch_logo.png

"Colors not applying"
  → sudo reboot

"Fastfetch not showing logo"
  → fastfetch
  → cat ~/.config/fastfetch/config.jsonc

"Waybar using old colors"
  → killall waybar && waybar &

For detailed troubleshooting, see ZENITSU_THEME_PLAN.md

CUSTOMIZATION OPTIONS
═══════════════════════════════════════════════════════════════════

Change Specific Colors:
  Edit: ~/.config/wallust/wallust.toml
  Section: [custom_colors]
  Then run: wallust run <your_wallpaper>

Use Different Wallpaper:
  Place new wallpaper in wallpapers directory
  Rerun: ./apply_zenitsu_theme.sh

Adjust Logo Size:
  Edit: ~/.config/fastfetch/config.jsonc
  Modify: "width" and "height" values

Revert to Original:
  Use timestamped backup files
  Or restore from JaKooLit defaults
