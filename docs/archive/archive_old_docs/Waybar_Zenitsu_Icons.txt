📝 HOW TO EDIT WAYBAR MODULES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. LOCATE WAYBAR CONFIG FILES:
   Main config: ~/.config/waybar/config.jsonc
   Modules: ~/.config/waybar/modules/

2. EDIT MODULE FILES:
   Each module (network, battery, etc.) has its own file.
   
   Example - Edit network module:
   $ nano ~/.config/waybar/modules/network.jsonc
   
   Look for "format" or "format-icons" sections:
   
   "format-wifi": "{icon}  {essid}",
   "format-icons": ["", "", "", "", ""]
   
   Replace the icons with Thunder Breathing themed ones:
   
   "format-wifi": "  {essid}",  // Lightning wifi
   "format-disconnected": "  Disconnected"

3. COMMON ICON LOCATIONS IN WAYBAR:
   • "format": The main display format string
   • "format-icons": Array of icons for different states
   • "format-charging": Battery charging state
   • "format-alt": Alternative display format
   • "tooltip-format": Tooltip icon/text

4. EXAMPLE MODULE CUSTOMIZATION:

   BATTERY MODULE (~/.config/waybar/modules/battery.jsonc):
   ────────────────────────────────────────────────────────
   "format": "{icon}  {capacity}%",
   "format-charging": "  {capacity}%",  // Thunder charge!
   "format-icons": ["", "", "", "", ""]

   WORKSPACES MODULE:
   ────────────────────────────────────────────────────────
   "format": "{icon}",
   "format-icons": {
       "active": "",    // Lightning for active
       "default": ""    // Circle for inactive
   }

5. RELOAD WAYBAR:
   After editing, reload Waybar to see changes:
   
   🔑 KEYBINDING: SUPER + ALT + R
   
   Or manually:
   $ pkill waybar && waybar &

🎨 COLOR COORDINATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Your Zenitsu theme uses these colors:
  • Haori Orange: #FFA237 (primary accent)
  • Haori Shadow: #B57436 (secondary)
  • Thunder Yellow: #FFE66D (highlights)

Add color to icons in Waybar CSS:
  ~/.config/waybar/style.css

Example:
  #battery.charging {
      color: #FFE66D;  /* Thunder Yellow for charging */
  }
  
  #workspaces button.active {
      color: #FFA237;  /* Haori Orange for active workspace */
  }
